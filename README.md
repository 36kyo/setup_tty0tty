# setup_tty0tty
setup shell script for tty0tty <https://github.com/freemed/tty0tty>

## how to use
All commands to execute are written in setup_tty0tty.sh

Set your path. Example:
```set your path.
FILEPATH=~/Downloads
FILENAME=tty0tty_1.2.tar.xz
```

Then,
```
$ sh setup_tty0tty.sh
```
